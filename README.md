## API Reference

#### Select Country

```http
  GET /api/?country=${country}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `country` | `string` | Your Desired Country |

#### Get item

```http
  GET /api/&category=${category}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| category      | `string` | Your Desired Category |
